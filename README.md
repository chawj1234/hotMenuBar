# hotMenuBar

> macOS 메뉴바 기반 생산성 앱

## 🎯 앱의 목표

- **앱 사용자 경험 향상**:  
  사용자가 앱을 사용하는 도중, 마우스 이동이나 윈도우 상단 메뉴바에 접근하지 않고도, 키보드 단축키(예: ⌘M)를 통해 macOS의 상단 메뉴바(파일, 편집, 포맷, 보기, 윈도우, 도움말)에 해당하는 주요 메뉴 항목들을 즉시 확인하고 조작할 수 있도록 한다.

- **페이지 내 메뉴 오버레이 구현**:  
  단축키 입력 시, 현재 보고 있는 앱의 페이지 중앙(또는 지정 위치)에 상단 메뉴바와 동일한 메뉴 구조가 팝오버, 오버레이, 혹은 시트 형태로 시각적으로 표시된다.  
  이 오버레이는 실제 macOS 메뉴바처럼 각 메뉴(예: 파일, 편집 등)를 그룹으로 묶어 보여주고, 각 그룹을 클릭하면 하위 메뉴(예: 새로 만들기, 복사 등)가 펼쳐진다.

- **메뉴 항목 직접 실행**:  
  사용자는 오버레이에 표시된 메뉴 항목을 마우스 클릭 또는 키보드(방향키, 엔터 등)로 직접 선택할 수 있다.  
  각 메뉴 항목은 실제 macOS 메뉴바와 동일하게 앱의 기능(예: 새 문서 생성, 복사, 붙여넣기, 보기 설정 등)과 연결되어 즉시 실행된다.

- **메뉴 상태 동기화**:  
  오버레이에 표시되는 메뉴 항목의 활성/비활성(Enable/Disable) 상태, 체크 표시 등은 앱의 현재 상태와 실시간으로 연동된다.  
  예를 들어, 복사할 내용이 없으면 "복사" 메뉴가 비활성화되는 등, 메뉴의 동작 가능 여부가 동적으로 반영된다.

- **접근성 및 일관성 유지**:  
  메뉴 오버레이는 macOS 표준 메뉴바와 최대한 유사한 구조, 단축키, 시각적 스타일을 유지하여, 사용자가 기존 macOS 사용 경험과 혼동 없이 자연스럽게 사용할 수 있도록 한다.

- **종료 및 전환 UX**:  
  메뉴 오버레이는 메뉴 외부 클릭, ESC 키, 또는 메뉴 항목 선택 시 자동으로 닫히며, 사용자의 흐름을 방해하지 않는다.

- **확장성 고려**:  
  향후 메뉴 항목 추가, 구조 변경, 다국어 지원 등도 쉽게 확장할 수 있도록 메뉴 데이터와 UI를 분리하여 설계한다.

## 📖 프로젝트 개요

hotMenuBar는 macOS 메뉴바의 주요 기능을 앱 내 오버레이로 재현하여, 사용자가 단축키 한 번으로 모든 주요 메뉴 기능에 빠르게 접근할 수 있도록 돕는 생산성 도구입니다. 익숙한 macOS 메뉴 경험을 유지하면서도, 앱 내에서 더욱 빠르고 직관적인 조작이 가능합니다.

## 🚀 주요 기능

### 현재 구현된 기능
- ✅ 키보드 단축키(⌘M)로 메뉴 오버레이 호출
- ✅ macOS 메뉴바와 동일한 메뉴 구조의 오버레이 UI
- ✅ 메뉴 그룹 및 하위 메뉴 트리 표시
- ✅ 마우스/키보드로 메뉴 항목 직접 실행
- ✅ 메뉴 활성/비활성, 체크 상태 등 실시간 동기화
- ✅ 접근성 및 macOS 일관성 유지
- ✅ 메뉴 외부 클릭, ESC, 선택 시 오버레이 자동 종료
- ✅ 메뉴 데이터와 UI 분리 설계(확장성/다국어 지원)
- ✅ 6개 메뉴 그룹 (파일, 편집, 포맷, 보기, 윈도우, 도움말)
- ✅ SF Symbols 아이콘 및 키보드 단축키 표시
- ✅ 호버 효과 및 스프링 애니메이션

### 계획된 기능
- 메뉴바에서 실제 시스템 메뉴 정보 가져오기
- 다국어 지원 (영어, 한국어)
- 사용자 커스텀 메뉴 추가 기능
- 메뉴 레이아웃 커스터마이징
- 설정 화면 및 환경설정
- 앱별 메뉴 구성 저장/복원

## 🔍 코드 구조

### hotMenuBarApp.swift
앱의 메인 진입점으로 SwiftUI의 `@main` 어노테이션을 사용하여 앱 생명주기를 관리합니다.

### ContentView.swift
앱의 주요 사용자 인터페이스를 담당하는 뷰입니다. 메뉴 오버레이와 메인 화면을 통합 관리하며, 사용자에게 앱 상태와 가이드를 제공합니다.

### Models/MenuItem.swift
메뉴 아이템과 메뉴 그룹의 데이터 구조를 정의합니다. 메뉴 타입, 액션, 키보드 단축키, 상태 등을 포함합니다.

### Services/MenuDataService.swift
메뉴 데이터를 생성하고 관리하는 서비스입니다. 앱 상태와 메뉴 동기화, 메뉴 액션 실행을 담당합니다.

### Views/MenuOverlayView.swift
macOS 스타일의 메뉴 오버레이 UI를 구현합니다. 메뉴 그룹 선택, 하위 메뉴 표시, 호버 효과, 애니메이션을 포함합니다.

### Managers/KeyboardShortcutManager.swift
글로벌 키보드 단축키(⌘M)를 처리하고 메뉴 오버레이 표시/숨김을 관리합니다.
